# Online CV

A simple, modern, and responsive online CV (Curriculum Vitae) created using only HTML and CSS. This project demonstrates clean web design principles and responsive layout techniques.

## Description

This is a personal online CV that showcases professional information including work experience, education, technical skills, languages, and certifications. The design features a modern gradient header, clean typography, and a responsive layout that works well on desktop, tablet, and mobile devices.

## Features

- **Responsive Design**: Adapts seamlessly to different screen sizes
- **Modern UI**: Clean and professional appearance with gradient accents
- **Semantic HTML**: Well-structured HTML5 semantic elements
- **Pure CSS**: No JavaScript dependencies, only HTML and CSS
- **Print-Friendly**: Can be easily printed or saved as PDF

## File Structure

```
CV/
├── index.html         # Main HTML file with CV content
├── styles.css         # CSS stylesheet with all styling
├── profile-photo.jpg  # Profile photo
└── README.md          # This file
```

## How to Use

1. Clone or download this repository
2. Open `index.html` in any modern web browser

### Adding Profile Photo

1. Save your profile photo in the CV folder
2. Name it `profile-photo.jpg` (or update the filename in `index.html`)
3. Recommended: Square format, minimum 300x300px for best quality
4. The photo will be automatically styled as a circular image in the header

### Styling

Modify `styles.css` to customize:
- Colors (currently using purple gradient theme)
- Fonts
- Spacing and layout
- Responsive breakpoints

### Color Scheme

The current color scheme uses:
- Primary gradient: `#667eea` to `#764ba2`
- Text: `#333`
- Background: `#f8f9fa`

You can easily change these colors in the CSS file to match your preferences.
